# Login-Systeme-Php

- ## Description :

This is an exemple of login systeme with PHP.

- ## Requirements :

  - Apache server
  - Install database
  - Config `functs_db.php`

- ## Config database :

  - Open `/api/includes/functs_db.php`

  - And edit with yours creditials : 
    ```
    "host" => "localhost",
    "dbusername" => "root",
    "dbpassword" => ""
    ```
